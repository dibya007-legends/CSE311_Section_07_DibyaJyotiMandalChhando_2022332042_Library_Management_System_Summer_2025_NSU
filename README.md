# Library Management System

## Description
Library Management System developed as part of the **CSE311 Summer '25** project (Section 07, **Dibya Jyoti Mandal Chhando**). This project helps librarians manage and track library operations efficiently. It allows seamless management of books, members, and transactions, making library management more effective and convenient.

### Key Features:
- Add, view, and manage books.
- Add, view, and manage members.
- Issue books to members and handle book returns.
- Track overdue books and calculate fines.
- Generate reports on popular books and highest paying customers.

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/your-repository.git
    ```

2. Navigate into the project directory:
    ```bash
    cd your-repository
    ```

3. Create a virtual environment:
    ```bash
    python3 -m venv venv
    ```

4. Activate the virtual environment:
    - For macOS/Linux:
      ```bash
      source venv/bin/activate
      ```
    - For Windows:
      ```bash
      venv\Scripts\activate
      ```

5. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

6. Set up your database and run the project.

## Usage

After installation, follow these steps to use the system:

1. Run the project:
    ```bash
    python app.py
    ```

2. Open your browser and visit `http://127.0.0.1:5000/` to start interacting with the system.

### Features:
- **Books Section**: Add and manage books, view all books, and edit existing book details.
- **Members Section**: Add new members, view member details, and edit.
- **Transactions Section**: Issue books to members, return books, and calculate fees for overdue books.
- **Reports Section**: Generate reports on most popular books and highest-paying members.
